Proyek ini adalah implementasi Natural Language Processing (NLP) untuk mengklasifikasikan sentimen teks menjadi tiga kategori: positif, negatif, dan netral. Menggunakan arsitektur Transformer untuk mempelajari pola makna dari data latih dan menerapkannya untuk melabeli data uji secara otomatis.
